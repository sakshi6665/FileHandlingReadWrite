This project is a simple Java program that demonstrates file handling.
It reads data from a text file, processes the content, and writes the result to another file.

The program counts the number of lines and words from the input file
and stores the processed output in a new file.

 What this project does-
- Reads data from a text file
- Counts total lines and words
- Writes the result to another file
- Handles file-related exceptions

 Concepts used-
- File handling in Java
- Scanner class
- PrintWriter
- Exception handling
- Loops and conditional statements

 How to run the program-
1. Open the project in IntelliJ IDEA
2. Make sure `input.txt` is present in the project root folder
3. Run the `Main.java` file
4. Check the generated `output.txt` file for results
